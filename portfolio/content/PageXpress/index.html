PageXpress

The lightest Content Management System ever.

Goal

Some small firms don't have a large amount of money to spend for their websites. That means That there's isn't much storage available and that databases are limeted or simply a no go. This is the scenario where PageXpress fits in.

PageXpress is based on a flatfile database system which is build from scratch. In spite of this low profile tech the CMS system if fully managable using the GUI.

Research

Needs

  • Content Management without any technical knowledge
  • A website specific website builder
  • An extremely light database system

Key Functions

  • A fast codeless pagebuilder build within a flexible framework
  • Intstant previewing of the changes made in the pagebuilder
  • Website specific data management modules
